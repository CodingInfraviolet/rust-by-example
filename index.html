    Introduction

    Variables

    Assignment

    let x = 1;

    1

    let x = 1;
x = 2;

    error[E0384]: cannot assign twice to immutable variable x

    let mut x = 1;
x = 2;

    2

    let x = "one";
let x = 2;

    2

    Variables are created as read-only by default. You can make them mutabel by using the mut keyword.

    Immutable variables can be shadowed using a second let statement. This reuses the variable name, while discarding the previous value.

    Types

    let x = 1_000;

    Type of x is an undetermined "integer"

    
let x : u64 = 18446744073709551615;

    Type of x is 64 bit unsigned int

    let x = 1_000_i32;

    Type of x is 32 bit signed int

    The type is inferred, but can be specified manually. Common types are signed integers (i8, i16, i32, i64, i128), unsigned integers (u8, u16, u32, u64, u128), and integers whose size depend on the system architecture and are used for working with memory (isize, usize).

    Other basic types are f32, f64, bool, char.

    Const

    const PI : f64 = 3.1415926535;

    3.141592653

    const TAU : f64 = PI * 2.0;

    6.28318531

    const G : f64 = 9.8_f64.ceil();

    error[E0015]: calls in constants are limited to constant functions, tuple structs and tuple variants

    Const variables are intended to never change. They can not use mut, and must include a type. They may also only be set to constant expressions.
